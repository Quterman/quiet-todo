# Tihiy Spisok

Small browser-based task planner with local storage.

## What it does

- Keeps tasks in Today, Later, Soft Day, and Done sections.
- Lets tasks move between sections by dragging.
- Tracks progress only for the Today goal.
- Stores data locally in the user's browser.

## How to run

Open `index.html` in a browser, or serve the folder with any static web server.

## Deploy

This app is static, so it can be deployed with GitHub Pages, Netlify, Vercel, or Cloudflare Pages.
